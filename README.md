## Sobre mí

Soy Marvin Ramírez, estudiante de Ingeniería de Software con una sólida formación técnica en Sistemas de Telecomunicaciones. Mi enfoque principal es el desarrollo web, donde domino HTML, CSS y JavaScript, transformando ideas en interfaces funcionales, responsivas y accesibles. También, me interesa mucho el diseño web y de aplicaciones, utilizando figma y canva para ello.

Además de mi experiencia en la web, tengo conocimientos en desarrollo de aplicaciones móviles con React Native y conocimientos de backend mediante Python, Java y C++. Me considero una persona adaptable, con gran capacidad de resolución de problemas y dispuesta a crecer profesionalmente.
